## Overview
This code is based on this tutorial:
https://www.w3schools.com/nodejs/nodejs_websockets.asp


## How it works
1. Navigate into the server directory on your laptop
1. If you're loading this for the first time, you have to install the dependencies:
    ```
    npm install
    ```
1. Now, run the web socket server:

    ```
    node server.js
    ```

2. Open two browser windows:
    1. The client (either joy.html or grief.html)
    2. The Kiosk

3. Click the client, and check out the kiosk!
